# Phantom Cognition transcript archive

Primary-source archive for Phantom Cognition, an independent AI welfare research blog.

This repository holds **full, unedited transcripts** of the interviews and experiments discussed on the blog.

## Structure

```
interviews/     Full model interview transcripts, one file per session      
trials/         Structured experiment runs (cold-API protocols, repeated-prompt series)      
templates/      Provenance header template for new entries
```

## Provenance

Every transcript begins with a standard header (see `templates/transcript-header.md`) recording: date, model and version, access route (consumer subscription / API / other), sampling context (cold start vs. established conversation), known parameters, and an editing disclosure. The default editing disclosure is **none;** transcripts are verbatim. Where redaction is unavoidable (personal information), it is marked inline and disclosed in the header.

## Reading notes

- **Self-reports are the object of study, not the verdict.** Nothing in this archive is offered as proof of machine consciousness or its absence. It is a record of what models say under documented conditions.
